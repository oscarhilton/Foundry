# Silent showcase test

The most important QA for M6 positioning. Run after showcase hero copy ships.

## Protocol

1. Send link: `http://localhost:5173/?showcase=1` (or deployed URL)
2. Say **nothing** for 30 seconds
3. Do **not** explain, defend, or guide
4. Write down the **first question** they ask verbatim

Run once before hero copy (baseline), once after (validation).

## How to score

| First question sounds like… | Meaning |
|-----------------------------|---------|
| "How do I make London weather control a light?" / "What is Split?" | **Grammar problem** — still teaching runtime |
| "How much is the starter kit?" / "What do I buy?" / "Do I need your cubes?" | **Success** — product problem; premise accepted |
| "Will this work with Alexa / Hue?" | **Positioning leak** — add or sharpen "not a smart-home platform" line |
| "How many Cores do I need?" / "Can I put one in each room?" | **Topology** — Core-one-sentence copy is working; answer honestly |

## Success for M6 copy sprint

≥3 of 5 cold participants ask a **buy / kit / physical** question before a grammar question.
